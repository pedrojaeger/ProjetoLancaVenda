<h1>Projeto Lança Venda </h1>

## Por quê? 

- Projeto de um sistema de controle de vendas do dia.
- CRUD com nome do cliente, valor da venda e onde entrou o pagamento.

## Conhecimentos / Habilidades treinadas e utilizadas
- Angular
- Angular Material
- TS
- HTML
- CSS
